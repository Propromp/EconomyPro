# EconomyPro
super-useful economy plugin
## features
- Full Vault support
- fixed API
- database support
- no junks. just economy.
## commands
- /money:check your balance
- /bank:manage your bank account
- /balance:manage balance
- /pay <player> <amount>:pay <player> <amount>
- /economypro:shows help
- /balancetop:displays Top 10
## permissions
- economypro.*
    default: op
- economypro.money
    default: all players
- economypro.bank
    default: all players
- economypro.balance
    default: op
- economypro.pay
    default: all players
- economypro.economypro
    default: all players
- economypro.balancetop
    default: all players
